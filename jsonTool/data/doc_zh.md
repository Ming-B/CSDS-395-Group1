# JSON 工具 — 使用说明

## 保存进度 与 保存 的区别
- **保存进度（快照）**  
  会在 `/workspace` 文件夹中生成一份 *临时快照*。  
  - 快照不仅记录当前的数据，还会记录 **界面的状态**（展开/折叠位置、光标位置）。  
  - 你可以通过 **撤回 / 重做** 在不同快照之间切换。  
  - 每次关闭文件或退出工具时，快照会自动清空。

- **保存（覆盖文件）**  
  会将编辑结果 **直接写回原始 JSON 文件**。  
  - 适合在确认修改无误后替换源文件。  
  - 这个操作没有快捷键，只能手动触发，以避免误操作。

## 特色功能
- **自动定位还原**  
  在撤回或重做时，工具会自动恢复到之前的浏览位置，包括展开层级、滚动条位置、当前选中条目。

- **支持编辑键和值**  
  在 *编辑器* 标签页中，你可以同时修改属性名（键）和属性值。  
  - 工具会避免同一个对象里出现重复键。  
  - 属性值会自动识别类型（字符串、数字、布尔、空值），并使用不同颜色显示。

## 标签页说明
- **拆分工具Splitter**
  不涉及快照/工作区：没有实现任何快照接口，仅仅显示和导出。
  左侧选中的子结构，点击右侧的确认按钮，可以导入任务列表。但支持 Ctrl/Cmd 多选、Shift 同父级范围选择（只选同一父节点下的连续“兄弟”行，不包含子孙节点）。右侧用于确认所选子结构、编辑导出名、删除条目，以及选择导出目录后批量导出为单独 JSON 文件。被选择的树节点以浅绿色高亮，直到点击“Confirm Selection”后清除。
  - 在左侧点击可以选择单个
  - 按住 ctrl 或 command，可以选择多个
  - 按住 shift 可以连续选择多个最外层子结构
  命名规则：
  - 若节点有字符串 key，直接用作名称；
  - 若是列表元素（key 为 int）且父节点有名字：{parent}_branch_{n}；
  - 父节点也没有名字：{NNNNNN}_branch_{n}，其中 NNNNNN 为全局 6 位自增编号；
  - 导出名默认 名称 + .json，可在表格第 2 列自行编辑；
  - 每行尾部 X 可删除该导出项。
